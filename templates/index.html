Advanced CTG Monitoring and AI Decision Support

Transparent Fetal-Net provides intelligent CTG analysis using machine learning and explainable AI to support fetal health monitoring.

The system analyzes fetal heart rate and uterine contractions to classify fetal condition into Normal, Suspect, or Pathological categories and provides transparent clinical explanations for healthcare practitioners.

Key Features

AI-Based CTG Analysis

Predict fetal health condition using highly accurate XGBoost machine learning models.

Explainable AI (SHAP)

Shows the exact important CTG features influencing each real-time AI prediction.

Active Learning Dashboard

Human-in-the-loop continuous learning paradigm improves model performance over time.

Clinical Reports

Generate formatted PDF medical reports and store patient CTG histories for monitoring.

Clinical Decision Support System

The system analyzes important detailed CTG clinical parameters such as:

  • Baseline fetal heart rate
  • Accelerations
  • Decelerations
  • Uterine contractions
  • Variability patterns

...and provides AI-supported clinical decision-making to detect fetal risk early.
